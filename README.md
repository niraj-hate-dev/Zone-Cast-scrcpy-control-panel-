# Zone Cast - Scrcpy Control Panel

## 📌 Overview
Zone Cast is a desktop application built using Electron.js that allows users to control and mirror Android devices using scrcpy.

## 🚀 Features
- Mirror Android screen on desktop
- Control device using mouse & keyboard
- Simple and lightweight UI

## 🛠 Tech Stack
- Electron.js
- Node.js
- HTML, CSS, JavaScript
- scrcpy

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/niraj-hate-dev/scrcpy-control-panel

2. Install dependencies:
   npm install

3. Run the app:
   npm start

## 📦 Build
To create executable:
npm run build

## ⚠️ Note
scrcpy is not included in this repository. Please download it separately and configure the path.

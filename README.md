# Zone Cast - Scrcpy Control Panel

## 📌 Overview
Zone Cast is a desktop application built using Electron.js that allows users to control and mirror Android devices using scrcpy.

## 🚀 Features
- Mirror Android screen on desktop
- Control device using mouse & keyboard
- Simple and lightweight user interface

## 🛠 Tech Stack
- Electron.js
- Node.js
- HTML, CSS, JavaScript
- scrcpy

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/niraj-hate-dev/Zone-Cast-scrcpy-control-panel-

2. Navigate to the project folder:
   cd scrcpy-control-panel

3. Install dependencies:
   npm install

4. Run the application:
   npm start

## 📦 Build
To create an executable file:
npm run build

## ⚠️ Note
scrcpy is not included in this repository. Please download it separately from the official source and configure the path before running the application.
